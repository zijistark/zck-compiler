# objects and random_* scopes which select 0-1 objects from a list of 0-N objects. It's currently used by token_codegen.py to
# generate token definitions and emitters for the lexer's specification file (zck.qx).

# NOTE: These are currently all *assumed* to be valid effects. It's possible that there are some weird trigger-only list
# scopes in here, and if so, they should be removed (or, rather, more fields need to be added to this file to differentiate
# trigger-only list scopes).
